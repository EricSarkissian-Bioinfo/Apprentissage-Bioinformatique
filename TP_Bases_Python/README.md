# Bio-informatique : Apprentissage et Algorithmes

Bienvenue sur mon dépôt dédié à mon apprentissage de Python dans le projet de préparer mon entrée en Master de Bioinformatique.


1) Nom du fichier : 01_calcul_imc_sante.py
 Objectif :  Calculer l'IMC d'un patient à partir d'un poids et d'une taille donné
 Concepts Python utilisés :
- utilisation des modes opératoires
- utilisation du type flottant (float)

2) Nom du fichier : sequence_adn_minuscule.py
   Objectif : Voir à quel type correspond la séquence ADN
   pour ensuite le mettre en minuscule
   Concept Python utilisés :
   -utilisation de type() pour voir quel type correspond la séquence ADN donnée
   -utilisation de isinstance(sequence_adn, type) pour verifier si la séquence ADN correspond bien au type
   - utilisation de sequence_adn.lower() pour mettre la séquence ADN donnée en minuscule
